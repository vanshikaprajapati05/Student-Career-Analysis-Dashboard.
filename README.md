# 🎯 Student Career & Skill Gap Analytics Dashboard

## Overview

The Student Career & Skill Gap Analytics Dashboard is a Power BI project designed to analyze student performance, technical skills, career preferences, placement readiness, and professional development activities. The dashboard provides actionable insights into skill gaps and employability factors through interactive visualizations and KPI-driven analysis.

## Objectives

* Evaluate student skill proficiency across key technical and professional domains.
* Identify skill gaps and areas for improvement.
* Analyze career preferences and expected salary trends.
* Assess placement readiness using academic and skill-based indicators.
* Examine the impact of internships, projects, and certifications on employability.

## Dataset

The dataset contains 500 student records with the following attributes:

* Student ID
* Gender
* Age
* CGPA
* Python Score
* SQL Score
* Power BI Score
* Excel Score
* Machine Learning Score
* Communication Score
* Projects
* Internships
* Certifications
* Career Path
* Expected Salary
